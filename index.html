Firebase সংযোগ

📋 ধাপ ১: Firebase প্রজেক্ট তৈরি করুন

  1. console.firebase.google.com এ যান
  2. নতুন প্রজেক্ট তৈরি করুন → নাম দিন
  3. Project Settings → General → Your apps → Web app (</>) যোগ করুন
  4. Config object কপি করুন, নিচে paste করুন
  5. Realtime Database → Create database → Start in test mode
⚠️ নিরাপত্তা: Realtime Database → Rules এ যান এবং rules পরিবর্তন করুন:
{"rules":{".read":"auth != null",".write":"auth != null"}}
অথবা ম্যানুয়ালি দিন ▼
